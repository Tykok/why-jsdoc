What's JSDoc

JSDoc is a documentation free tool for JavaScript (created in 1999), it's a markup language to annote many things in your JavaScript source code. It is a simple way to document your code.

You can simply use it to describing the program you wrote and it's also used by some librarys to generate documentation in other format like HTML for example.
Today, JSDoc is also implemented into many IDE.
And each other...

If JSDoc was implemented in many IDE, it's for a really good reason and i'll show you why after.

Disadvantages

Advantages

Specifies many important information like the type of a variable for example.

Makes reading faster for YOU, for OTHERS PEOPLES, and... for your IDE (yes you're IDE use JSDoc to make autocompletion)
